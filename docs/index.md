# Get Started

## Welcome to Syncwave

Turn plain JSONs into a live data store, two-way synced with Python objects.

## Installation

Install from [PyPI](https://pypi.org/project/syncwave/).

=== "pip"

    ```bash
    pip install syncwave
    ```

=== "uv"

    ```bash
    uv add syncwave
    ```

Requires Python 3.9+.
